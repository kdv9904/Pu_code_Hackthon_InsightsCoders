You need the following installed on your computer:

üì• INSTALLATION STEPS
================================================================================

Step 1: Open Terminal/Command Prompt
-------------------------------------
- Windows: Press Win + R, type "cmd", press Enter
- Mac: Press Cmd + Space, type "terminal", press Enter
- Linux: Press Ctrl + Alt + T


Step 2: Navigate to Project Folder
-----------------------------------
cd react-native

(Replace "react-native" with the actual path if different)


Step 3: Install Dependencies
-----------------------------
npm install

This will take 2-5 minutes depending on your internet speed.
You should see a progress bar and installation messages.


Step 4: Start Development Server
---------------------------------
npm start

You will see:
- QR code displayed in terminal
- Message: "Metro waiting on exp://..."
- Options menu with keyboard shortcuts


üöÄ RUNNING ON ANDROID
================================================================================

METHOD 1: Using Android Emulator
---------------------------------
1. Make sure Android Studio is installed
2. Open Android Studio ‚Üí AVD Manager
3. Start your Android Virtual Device (emulator)
4. Wait for emulator to fully boot
5. In the terminal where you ran "npm start", press: a

The app will install and launch automatically.


METHOD 2: Using Physical Android Device
----------------------------------------
1. Download "Expo Go" from Google Play Store
2. Connect your phone to the SAME WiFi as your computer
3. Open "Expo Go" app
4. Tap "Scan QR Code"
5. Scan the QR code shown in your terminal

The app will load on your device.


‚ö†Ô∏è TROUBLESHOOTING
================================================================================

Problem: "Command not found: npm"
Solution: Install Node.js from https://nodejs.org/

Problem: "Metro bundler won't start"
Solution: 
  expo start -c
  (This clears the cache)

Problem: "Cannot connect to development server"
Solution:
  - Make sure computer and phone are on same WiFi
  - Disable firewall temporarily
  - Try: expo start --tunnel

Problem: Android emulator not detected
Solution:
  - Make sure emulator is running BEFORE pressing 'a'
  - Check if ANDROID_HOME environment variable is set
  - Restart Android Studio

Problem: Dependencies installation failed
Solution:
  rm -rf node_modules
  rm package-lock.json
  npm install

üèóÔ∏è BUILDING APK FOR DISTRIBUTION
================================================================================

To create an APK file for installation on any Android device:

1. Install EAS CLI:
   npm install -g eas-cli

2. Login to Expo:
   eas login

3. Build APK:
   eas build -p android --profile preview

4. Download APK:
   You'll receive a link to download the APK file

5. Install APK:
   Transfer APK to Android device and install
